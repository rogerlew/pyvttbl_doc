Marginals¶

Used to calculate marginal descriptive statistics.

Example¶

To calculate marginals we first need to get data into a DataFrame

>>> df=DataFrame()
>>> df.read_tbl('data/words~ageXcondition.csv')
>>> D = df.marginals('WORDS',factors=['AGE','CONDITION'])
>>> print(D)
